# 🎬 Favorite Movie Manager (Python)

A simple command-line Python application to manage your favorite movies.  
This project allows users to add, view, and remove movies from a list in an interactive way.

## 🚀 Features
- 📌 Add multiple favorite movies
- 📋 View all saved movies
- ❌ Remove a movie from the list
- 🔁 Continuous menu-driven interface
- ⚠️ Input validation for better user experience

## 🛠️ Technologies Used
- Python (Core Concepts)
- Sets (for storing unique movie names)
- Loops & Conditional Statements

## 📷 How It Works
The program runs in a loop and displays a menu:
1. View Movies  
2. Add Movies  
3. Remove Movie  
4. Exit  

## ▶️ How to Run
```bash
python movie_manager.py
